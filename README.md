<p>exception-boundary是一个容器，当errorMsg是undefined时渲染子元素，当errorMsg是一个字符串时渲染这个字符串，并将这个字符串显示到浏览器控制台。当控件需要在某些条件下（比如，重复设置某个属性），在页面显示一个异常信息时，可以使用exception-boundary。</p>
